Το project αναπτύχθηκε με τον IDE IntelliJ IDEA και με sdk 1.8. Περιέχει 2 packages τα οποία περιέχουν μία main η οποία τρέχει το
πρόγραμμα. Για να τρέξει χρειάζεται απλα να ανοιχτεί απο το IntelliJ.
Για το Missionaries And Carnivals problem:
Θα πρέπει να θέσετε τις τιμές N,M στο new State line 8 (έχει default 3,2) και την τιμή K στο crossingsAllowed line 9 (έχει default 1000)
Για το N Queens Problem:
Θα πρέπει να θέσετε την τιμή N στο g.geneticAlgorithm line 8 (έχει default 8)
